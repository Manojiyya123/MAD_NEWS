# NewsNow - Android News App

NewsNow is a simple Android application that displays a list of news articles with a clean and responsive UI.

## Features
- **News List**: Browse through a curated list of news articles.
- **Responsive Layout**: Adapts for both phones and tablets (GridLayout on tablets, LinearLayout on phones).
- **Refresh Content**: Easily refresh the news feed from the toolbar.
- **Connectivity Awareness**: Basic handling for network state changes.
- **Modern UI**: Uses Material Design components like CardView, RecyclerView, and Toolbar.

## Technologies Used
- Java
- Android SDK (Min SDK 24, Target SDK 36)
- Material Components for Android
- RecyclerView & CardView
- ConstraintLayout

## Getting Started
1. Clone this repository.
2. Open the project in Android Studio.
3. Build and run on an emulator or a physical device.
